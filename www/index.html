Fast and lightweight
  • dred is built as a single executable. Nothing more. Just drop it in and run.
  • dred is small. It's executable is less than a megabyte and it uses less memory than most other popular text editors.
  • dred is fast, with virtually instant loading. Stop wasting time waiting for things to load and increase productivity.
Multi-edit and multi-select
  • Stop making the same change many times - just do it all in one go.
  • Just hold down CTRL and select text as you normally would. Easy and intuitive.
Customizable
  • dred has a simple but powerful configuration system which allows you to change most settings. More settings are being added all the time!.
  • Loading dred from a directory with a .dred file will automatically load the config file.
  • Config files can be loaded at run time with the "load-config" command.
